## Stack Details

1. *Ruby version: (2.5.1)*
2. *Rails version: (6.1.0)*
3. *Puma (5.1.1)*

## Instruction to run the app

- *rails db:create*
- *rails db:migrate*
- *rails s*
- *Navigate to the Help section, to know the list of valid feed urls.*
- *Adding a feed url, would pull posts automatically in the background.*
